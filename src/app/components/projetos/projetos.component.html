Site de arrecadação de dinheiro para presente

Este é um site que permite a criação de campanha de arrecadação para presentear alguém em ocasiões especiais, a interface do usuário facilita a doação, exibindo o valor arrecadado, a meta a ser atingida e o número de apoiadores. Os usuários podem visualizar a descrição do presente e contribuir diretamente pela plataforma que gera um QR code com a chave pix do organizador e o valor a ser doado escolhido pelo usuário. . A stack utilizada inclui Laravel no backend para gerenciar as APIs e JavaScript puro (Vanilla JS) no frontend para a interação do usuário. O site pode ser acessado via navegador e possui um design intuitivo para facilitar a experiência dos usuários.
