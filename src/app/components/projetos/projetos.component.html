Site de vaquinha online

Descrição:Plataforma web de arrecadação coletiva (vaquinha online).
Funcionalidade:Exibe o total arrecadado em relação à meta com contador e barra de progresso, permite registro de doações com nome e valor, gera QR Code PIX automaticamente para pagamento, mostra o número total de contribuintes e detalha os custos para obtenção da CNH.
Tecnologias: React (TypeScript): Construção da interface de usuário; \n Vite: Ferramenta de build rápida. Supabase (PostgreSQL): Armazenamento das doações. QRCode: Geração de códigos PIX. React Toastify: Notificações interativas. Hotjar: Análise de comportamento do usuário. CSS: Estilização e animações.

Encontro de Noivos - Plataforma de Sorteio e Acompanhamento

Descrição: Plataforma web para gestão de encontros de preparação para o matrimônio, com sorteio automático de casais de noivos para casais mentores.

Tecnologias: React (TypeScript) Vite React Router Material UI Supabase (PostgreSQL) CSS
